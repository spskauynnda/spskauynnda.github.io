1. 匿名函数是什么？

初学者（比如我）不懂匿名函数和闭包，常混为一谈，二者其实截然不同。什么是匿名函数？先对比下面两段函数声明：

上面两段都是函数的一般声明方法，在逻辑上几乎是等价的，它们有两个微小的区别：

  1. 前者在执行前就被加载到作用域中，后者执行到该行才有定义
  2. 前者为函数指定了函数名FUNC（指针），后者则是今天的重点——将匿名函数赋给了FUNC变量

匿名函数，即没有指定变量名的函数，形如 function ( ) { … }

本篇的主角，就决定是你了。

3. 闭包是什么？

闭包，即访问另一函数作用域中的私有变量

实现原理：作用域链
