华辩自由辩技术分析——记2017新国辩初赛 “被同化/被排斥更可怕” （复旦/天大）

前言：写得比较乱，边听比赛边结合分析看可能食用更佳。这场其实是一个标准争夺战，只是正方没明说，而反方本身论证缺环也不敢说。

一、 实况

正方判标：谁能让糟糕的现状更糟糕

反方判标：谁本质上造成了伤害（本质这个词容易产生歧义，按我的理解其实是说谁【必然引发】了伤害的产生）

1. 正方第一轮进攻

正四：先来看您方说的本质伤害，我方反驳过排斥不一定引发痛苦，另外排斥引发的对立也可能促进争议和讨论，是不是可能促进进步？（挑战反方核心论点）
反三：不，无论变革结果是好是坏，都会引发战争，战争是可怕的
正三：所以是否一定痛苦您方论证不出 / 所以您方论证不出痛苦是否根属于被排斥（追问题，继续挑战）
反四：1.那可能是因为大脑有一种自我保护机制，感受到痛苦以后骗自己：这不是痛苦（——感觉是逃问题，正方挑战的理由在于被排斥也可能有正面效应，他的回应是痛苦可能不是痛苦）2.同化本身可不可怕？是不是其实只是个增幅作用呢？（开启反方的主攻战场）
正一：所以排斥一定有痛感我方已经切掉了（总结战场，同时逃了反方的问题）再请问你豫章书院这种同化是不是也会有痛感，二者都有痛感您方怎么比？（强化进攻，开始进攻反方的比较标准——被排斥是绝对痛感）
反一：被排斥不痛苦可能只是心理学上的自我欺骗（延续反四的解释，硬着头皮给出不痛苦本质可能还是痛苦的一种解释，但回应得很疲软）
正四：（提出两种例外）1.君子道不同不相为谋是否痛苦 2.被讨厌的人排斥是否痛苦，有时候甚至产生优越感引发自我的改进。因此为什么是本质上的伤害？（这里看起来好像和正方只讨论负面作用的标准相悖，但其实是在打反方的比较标准——谁是本质伤害谁就更可怕。由于这里没有直接拆反方标准以至于反方还感觉不够痛。另外我觉得其实还有更强的拆法，但拆到这里基本把反方的立论拆完了）
反四：您方一直在说排斥本质上不是伤害，那您方怎么论证被同化本质上有什么伤害？（面对正方这一段非常强的挑战，反过来质疑正方是否论证出了本质伤害。第一没有正面回应挑战，第二说他傻也巧妙，拿己方标准去证明对方论证失利……要是正方直接进攻反方的标准这里就显得傻了）

2. 正方第二轮进攻

正三：（开始隐约戳双方标准的差异）无论是被同化还是被排斥都没有直接和伤害挂钩（在对方的标准底下比不出谁有本质伤害），所以我们比较在具体的负面的情境底下谁让现状更可怕（回顾己方标准）我们来比较本质：被邪教排斥很危险，但还能找到别人的协助甚至打击邪教，如果被邪教同化了，不仅自己会受害，还会一起祸害他人，这样扩展可怕是不是本质上更可怕？（扎实的攻击，看得出是事先准备的战场问题）
反一：那没有被邪教同化的人集结起来排斥邪教组织，这可怕吗？所以您方不能论证被同化必然可怕（这个总结很强势，但理由很奇怪而且还是在逃正方的问题）
正一：去除所有手段，本质上就是两种价值观的对立，这有什么可怕？本质上看一者是摆脱，一者是助纣为虐，谁更可怕？（这里开始反方马上就要被打穿了）

3. 正方开始照着标准推例子，反方第一轮反攻尝试

反一：（干脆不接问题了，换战场）所以被同化本质上只是做乘法？所以本身并不可怕是吗？
正三：您这样理解就错了，您方本质上论证不出排斥和伤害挂钩，所以我们【只能去比】谁在可怕的情境下谁的因素更大（能让伤害扩大/其实直白来讲就是他方标准不成立，按我方标准比，但按他的表述其实没讲透这个意思）
反四：被邪教排斥可怕吗？
正一：可怕，但是我能意识到，能采取措施去应对，瓦解邪教，而如果被同化，我都意识不到这是坏的，甚至助长邪教势力，是不是更可怕？
反四：被邪教排斥可怕在哪里？
正一：摧毁精神
反四：所以您说的可怕是在社会层面是吗
正四：邪教的例子您不回应，抗战中被敌军排斥很危险，那如果被敌军同化了反过来残害同胞壮大势力是不是更可怕？
反二：您方举的例子不考虑方向性，那被共产党同化是不是就不好呢？（脑补：被同化有时候是好的有时候不好）
正一：在抗日战争中balabala……（反方把脖子送过来了，她这里回应错了，应该跟着己方三四辩拆，被排斥也可能会有好的方向，所以应该比较谁能让伤害更大/这里双方一辩都有点迷）

4. 反方反攻亮明底线

反四：所以您方不考虑方向性，是不是被同化只是增幅器，也可以导出光明呢？
正三：考虑方向性两者都比不出（bingo）被同化是不是更能扩大伤害？
反一：小芬被排斥可不可怕？
正四：您方的可怕也是用方向性推导出来的
反三：我方认为对抗就一定有一方受到伤害（总算有论证为什么一定有本质伤害了，但一下就把底线暴露出来了），如果没有一方受到伤害社会就一直停步了（？？后面这句很迷，给对方出了题还附送答案的吗？）
正二：政见意见的对抗，美学学术流派的对抗，君子割袍断义道不同不相为谋，志趣不同老死不相往来，这些排斥一定有伤害吗？（1. 太赞，最后十几秒挑战了对方刚暴露出的底线 2. 如果能在开头强调一下“对方终于暴露出他们本质性论证的前提了”就更好了，在自由辩论的最后往往评委很容易头晕记不住你的话）
反方30秒缺席审判：被排斥一定互相打压产生伤害

二、总结

  1. 正方没有强化己方的攻击其实是在拆对方的标准，以至于反方质问正方是不是增幅器，正方解释得很薄弱，其实就一句话：你的标准下比不出，因为你也是增幅器，大家一起来比谁增幅效果更强才是正理。
  2. 自由辩要提前准备战场问题，问题要准备多重表述，否则总是问“被同化是不是更可怕”会显得毫无进攻压力
  3. 追问题要像正方一样持续追，接战场一定要迅速果断，并且不要留回应死角
    （1）要是知道自己这里有死角千万不要把战场主动权留给对方，主动去进攻逼迫对手解释自己的立论问题才行。
    （2）对于死角要提前准备，或强硬防守（解释，用定义去切割讨论范围），或用受身承认例外
  4. 理清楚哪里是我方的利益点，哪里是对方的利益点，拉平只能用在对方的论点上，不能拉平自己的。比方，我说因为我鼻子很挺所以我比你帅，对方说你鼻子不挺啊，我拉平：可是可是……你鼻子也不挺啊！……就会显得很蠢了
  5. 拆点要拆到位，否则会让人觉得没打穿没讲清楚。拆例子要拆到逻辑，拆逻辑要拆到论点，拆论点要拆到对方的立论底线（判断标准，或是基本逻辑前提，一般都是双方的根本分歧点）；强化攻击收割利益，可能评委一帧一帧回顾脑补会觉得你很有道理，但是自由辩这种紧张刺激的环节很可能会漏点，必须时刻提醒评委记点

三、思考

  1. 这场自由辩逐帧分析算是正方完胜，但是实际观感会觉得势均力敌甚至有人觉得反方自由辩小优，这是为什么？是他理解错了吗？
  2. 正方应该做哪些处理能够让自由辩能打下来？
  3. 反方哪里存在根本问题，应该在赛前和自由辩中做哪些改进？
  4. 你觉得一个成功的自由辩中，一到四辩应该如何分工？应该如何布置战术？
  5. 自由辩好坏的标准是什么？自由辩在整场比赛中的意义是什么？
  6. 你觉得这篇实况分析有错漏吗？错在哪儿？
